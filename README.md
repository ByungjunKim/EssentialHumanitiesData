## 디지털 인문학과 디지털 에디션: 데이터 활용 사례

## 디지털 인문학 개요 및 데이터 리터러시

디지털 인문학(Digital Humanities)은 인문학 연구에 디지털 기술과 컴퓨팅 기법을 접목하여 새로운 연구 방법과 질문을 탐색하는 학제 간 분야입니다. 문헌, 역사, 철학 등 전통적인 인문학 자료를 데이터로 변환하고, 컴퓨터를 이용해 처리·분석함으로써 인간 문화에 대한 새로운 통찰을 얻는 것을 목표로 합니다.  이러한 맥락에서 **데이터 리터러시(data literacy)**, 즉 데이터를 접근, 평가, 관리·분석하고 윤리적으로 활용하는 능력은 매우 중요합니다. 디지털 인문학 프로젝트 및 교육 과정에서는 정성적, 정량적 방법을 **융합하여** 데이터를 다루며, 학생과 연구자들의 데이터 활용 역량 향상을 목표로 합니다.

## 인문 데이터와 디지털 에디션

### 1. 기계가독형 데이터와 디지털 에디션의 중요성

디지털 시대의 인문학 연구는 새로운 도구와 방법론을 필요로 합니다. 단순히 텍스트를 디지털화하는 것을 넘어, 컴퓨터가 이해하고 분석할 수 있는 형태로 데이터를 구조화하는 것이 중요해졌습니다.  **기계가독형 데이터**는 디지털 에디션의 핵심이며, 텍스트 마이닝, 의미 분석, 시각화 등 다양한 디지털 인문학적 분석을 가능하게 합니다.

김동인의 소설 「감자」를 예시로, 인문학 연구에서 활용되는 주요 데이터 형식들을 살펴보고, 디지털 에디션을 위한 데이터 처리 과정을 알아봅니다.

### 2. 인문학 데이터 파일 형식

#### 2.1 텍스트 파일 (TXT)

- 가장 기본적인 텍스트 저장 방식
- 서식이나 특별한 구조가 없는 순수 텍스트
- 용량이 작고 호환성이 뛰어나며, 디지털 데이터화의 **기본 단계**가 됨

```txt
[감자]
작가: 김동인
발표: 1925년
장르: 단편소설

싸움, 간통, 살인, 도둑, 구걸, 징역, 이 세상의 모든 비극과 활극의 근원지인, 칠성문 밖 빈민굴로 오기 전까지는...
```

#### 2.2 CSV/TSV 파일 (표 형식 데이터)

- CSV(Comma-Separated Values): 쉼표로 구분된 데이터
- TSV(Tab-Separated Values): 탭으로 구분된 데이터
- 행과 열로 구성된 구조화된 정보, **테이블 형태** 데이터
- 스프레드시트 프로그램과 호환성이 좋으며, **데이터 분석 및 시각화**에 용이함

```csv
이름,신분,나이,성별,직업,성격특성,거주지,가족관계
복녀,농민,28,여성,소작농,"순박함,성실함,순종적",산골 마을,"남편(정선달-사망), 자녀(딸-사망)"
김 주사,지주,50,남성,지주겸 고리대금업자,"탐욕스러움,교활함,이기적",산골 마을,없음
```

```tsv
장	사건	시기	장소	관련인물	내용	결과
1	남편과 딸의 죽음	이야기 시작 전	산골 마을	복녀,정선달	남편과 딸이 병으로 사망	복녀 혼자 남음
2	김 주사의 착취 시작	봄	감자밭	복녀,김 주사	소작료 인상 요구	복녀의 생활고 심화
```


#### 2.3 JSON 파일 (JavaScript Object Notation)

- 계층적 구조를 가진 데이터 표현 방식, **트리 형태** 데이터
- 객체와 배열을 통해 복잡한 관계 표현 가능
- 웹 환경에서 널리 사용되며, **데이터 교환 및 웹 기반 시각화**에 적합함
- 사람이 읽고 쓰기 쉬운 형식

```json
{
  "작품": {
    "제목": "감자",
    "저자": "김동인",
    "출판년도": 1925,
    "주요_등장인물": [
      {
        "이름": "복녀",
        "신분": "농민",
        "나이": 28,
        "성격특성": [
          "순박함",
          "성실함",
          "순종적"
        ],
        "변화과정": [
          {
            "단계": "농사꾼의 아내",
            "상태": "가난하지만 성실한 삶"
          },
          {
            "단계": "과부",
            "상태": "생계를 위한 고군분투"
          }
        ]
      }
    ]
  }
}
```


#### 2.4 XML/TEI 파일 (Text Encoding Initiative)

- 텍스트의 구조와 의미를 표현하는 마크업 언어, **구조화된 텍스트 데이터**
- 인문학 연구, 특히 **디지털 에디션**에서 널리 사용되는 표준 형식
- **텍스트의 의미와 구조**를 명확히 표현하여, 깊이 있는 분석과 다양한 활용 가능
- 다양한 분석 도구와의 호환성이 높음

```xml
<?xml version="1.0" encoding="UTF-8"?>
<TEI xmlns="http://www.tei-c.org/ns/1.0">
  <teiHeader>
    <fileDesc>
      <titleStmt>
        <title>감자</title>
        <author>김동인</author>
        <respStmt>
          <resp>TEI 인코딩</resp>
          <name>디지털 인문학 연구소</name>
        </respStmt>
      </titleStmt>
      <publicationStmt>
        <publisher>개벽</publisher>
        <date>1925</date>
      </publicationStmt>
    </fileDesc>
  </teiHeader>
  <text>
    <body>
      <div type="character">
        <person xml:id="BOKNYEO">
          <persName>복녀</persName>
          <trait type="personality">
            <label>성격</label>
            <desc>순박함</desc>
          </trait>
          <residence>산골 마을</residence>
        </person>
      </div>
    </body>
  </text>
</TEI>
```

## 디지털 에디션의 정의와 필요성

**디지털 에디션(digital edition)**은 전통적인 종이 기반 학술판을 디지털 환경에 구현한 결과물입니다. 단순히 텍스트를 스캔하여 PDF로 만드는 것을 넘어, 디지털 매체의 장점을 적극 활용하여 **인쇄 매체로는 불가능한 풍부한 콘텐츠와 기능**을 제공합니다.

**디지털 에디션의 필요성**:

- **풍부한 읽기 경험**: 멀티미디어, 하이퍼링크, 동적 탐색 기능 제공
- **심층적 연구 지원**: 원문에 대한 신뢰성 있는 접근, 다양한 주석과 메타데이터 포함
- **지식 전달 혁신**: 전통적 출판 방식의 한계 극복, 새로운 연구 및 지식 공유 방식 제시

## 디지털 에디션을 위한 데이터 처리 과정

디지털 에디션은 체계적인 데이터 처리 과정을 거쳐 완성됩니다.

1. **실물 스캔 및 OCR**: 종이 문헌 스캔 후 OCR 기술로 텍스트 데이터 추출
2. **TXT 생성**: OCR 결과 정제 및 순수 텍스트 파일(.txt) 저장 (**1차 데이터화**)
3. **Word 문서 제작 (선택적)**: 서식, 문단 구분 등 추가 (가독성 향상)
4. **TEI/XML 변환**: 텍스트 내용을 TEI(XML) 표준에 맞춰 태깅 (**구조화된 데이터**)
5. **XML 파싱 및 데이터 추출**: TEI/XML 데이터에서 필요한 정보 추출, CSV/JSON 등 분석 친화적 형식으로 변환 (**분석 및 활용 용이**)

### 3. 디지털 에디션의 활용 및 장점

#### 3.1 디지털 에디션의 개념 (재강조)

디지털 에디션은 텍스트를 **구조화**하고 **표준화**하여 컴퓨터가 인식하고 분석할 수 있는 **기계가독형 데이터**로 만드는 핵심적인 디지털 인문학 작업입니다.

#### 3.2 디지털 에디션의 장점

- **데이터 분석 용이성**: TEI/XML 구조화 데이터는 텍스트 마이닝, 단어 빈도 분석, 네트워크 분석 등 다양한 분석에 효율적으로 활용 가능
- **판본 비교**: TEI 태깅을 통해 프랑켄슈타인 판본 비교처럼, **텍스트의 변화 과정** 분석 용이
- **데이터 재사용성**: TEI, CSV 등 표준 포맷은 연구 데이터 공유 및 다양한 분석 도구 활용에 유리
- **교육 효과**: TEI 태깅 및 데이터 추출-분석 실습을 통해 학생들의 **데이터 리터러시** 및 **디지털 분석 역량** 향상

#### 3.3 연습문제: Raw Text를 디지털 데이터로 만들기

**예시 Raw Text**:
조선왕조실록은 우리 민족의 역사를 생생하게 기록한 귀중한 자료입니다. 이 기록은 각 왕의 즉위, 치세, 그리고 다양한 사회·문화적 사건들을 상세히 담고 있습니다.

**연습문제**:
1. TXT 파일 저장 및 기본 전처리 (**TXT 데이터 생성**)
2. TEI/XML 변환 및 사용자 정의 태그 적용 (**XML 데이터 구조화**)
3. Python으로 특정 태그 내용 추출 및 CSV 파일 저장 (**CSV 데이터 변환**)
4. 추출된 CSV 데이터 기반 단어 빈도 분석 또는 시각화 (**데이터 분석 및 활용**)

#### 3.4 사례 연구: 디지털 에디션 실제 활용

##### 3.4.1 Women Writers Project (WWP)

[Women Writers Project](https://wwp.northeastern.edu/)는 16-19세기 여성 작가 텍스트 TEI 인코딩 온라인 컬렉션 제공. TEI 태깅으로 서지 정보, 본문 구조, 원문 형태/편집 기록 보존, 고급 검색 및 분석 도구 제공. **디지털 에디션의 대표적 사례**

Frankenstein 여러 판본 TEI 태깅 비교 분석. 판본 간 차이, 수정/추가/삭제 텍스트 `<add>`, `<del>` 등으로 표시. **판본 비교 디지털 에디션의 효용성** 입증

## 4. 파이썬 코드 예시: 다양한 파일 형식 데이터 활용

```python
import pandas as pd
import json
import xml.etree.ElementTree as ET
import matplotlib.pyplot as plt

# CSV 파일 읽기 및 내용 확인 (pandas DataFrame 활용)
df_characters = pd.read_csv('characters.csv')
print("CSV 파일 내용 (등장인물):")
print(df_characters.head().to_markdown(index=False, numalign="left", stralign="left"))

# TSV 파일 읽기 및 내용 확인 (pandas DataFrame 활용)
df_events = pd.read_csv('events.tsv', sep='\t')
print("TSV 파일 내용 (주요 사건):")
print(df_events.head().to_markdown(index=False, numalign="left", stralign="left"))

# JSON 파일 읽기 및 정보 추출 (json 라이브러리 활용)
with open('gamja.json', 'r', encoding='utf-8') as f:
  data = json.load(f)
print("JSON 파일 내용 (작품 정보):")
print(f"제목: {data['작품']['제목']}")
print(f"저자: {data['작품']['저자']}")
print(f"출판년도: {data['작품']['출판년도']}")

# XML 파일 읽기 및 정보 추출 (xml.etree.ElementTree 활용)
tree = ET.parse('gamja.xml')
root = tree.getroot()
ns = {'tei': 'http://www.tei-c.org/ns/1.0'} # TEI 네임스페이스 정의
title = root.find('.//tei:title', ns).text
author = root.find('.//tei:author', ns).text
print("XML 파일 내용 (작품 정보):")
print(f"제목: {title}")
print(f"작가: {author}")

# 데이터 시각화 예시 (pandas, matplotlib 활용)
character_status_counts = df_characters['신분'].value_counts()
character_status_counts.plot(kind='bar')
plt.title('등장인물 신분별 빈도')
plt.xlabel('신분')
plt.ylabel('빈도수')
plt.show()
```

## 디지털 인문학 연구에서 데이터 활용의 중요성

디지털 인문학 연구에서 **데이터 활용 능력**은 필수적입니다. 방대한 인문학 자료를 효과적으로 분석하고 새로운 통찰력을 얻기 위해서는 **데이터 기반 접근 방식**이 중요하며, **디지털 에디션**은 이러한 접근의 핵심적인 결과물입니다. 디지털 에디션과 다양한 데이터 분석 및 시각화 기법을 통해 인문학 연구는 더욱 깊이 있고 혁신적인 방향으로 발전할 수 있습니다.
